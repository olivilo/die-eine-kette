# ⛓️ Die Eine Kette (DieEineKette)

> ⚠️ **公开 Alpha 版。** 这**仅是 Die Eine Kette 的免费公开部分**，目前**正在积极开发中**——
> 可能存在缺陷、功能未完成且随时变动。**尚未可用于生产环境。** 欢迎反馈与提交 issue。

> **一扇门，连接一切。**
> 一个多租户（B2B）大模型网关，全面多语言支持，对*外购*和*自托管*令牌进行成本控制，并配有现代化界面。

Die Eine Kette 将任意数量的大模型提供商（OpenAI、Anthropic、Gemini、Azure、AWS Bedrock、
Ollama、DeepSeek、Mistral、本地模型……）整合到**一个**兼容 OpenAI 的 `/v1` 接口之后，
并在其上叠加面向企业、部门、用户与预算的 B2B 管理层。

> 🚧 公开 Alpha —— 参见 [`docs/`](./docs)。启动：`docker compose up`。
