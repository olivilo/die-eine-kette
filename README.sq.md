# ⛓️ Die Eine Kette (DieEineKette)

> ⚠️ **Alfa publike.** Kjo është **vetëm pjesa falas dhe publike** e Die Eine Kette dhe është
> **në zhvillim aktiv** — pritni gabime, veçori të papërfunduara dhe ndryshime. **Ende jo
> gati për prodhim.** Reagimet dhe raportimet janë të mirëseardhura.

> **Një portë për t'i lidhur të gjitha.**
> Një portë LLM shumë-qiramarrëse (B2B), plotësisht shumëgjuhëshe, me kontroll të kostos
> mbi token-at *e blerë* dhe *të vetëpritura*, dhe një ndërfaqe moderne.

Die Eine Kette bashkon një numër të pakufizuar ofruesish LLM (OpenAI, Anthropic, Gemini,
Azure, AWS Bedrock, Ollama, DeepSeek, Mistral, modele lokale …) pas **një** ndërfaqeje të
vetme `/v1` të pajtueshme me OpenAI, me një shtresë menaxhimi B2B sipër (ndërmarrje,
departamente, përdorues, buxhete).

> 🚧 Alfa publike — shih [`docs/`](./docs). Nisja: `docker compose up`.
