# ⛓️ Die Eine Kette (DieEineKette)

> ⚠️ **Javna alfa.** Ovo je **samo besplatni, javni dio** projekta Die Eine Kette i **u
> aktivnom je razvoju** — očekujte greške, nedovršene funkcije i izmjene. **Još nije spremno
> za produkciju.** Povratne informacije i prijave su dobrodošle.

> **Jedna kapija da ih sve poveže.**
> Višekorisnički (B2B) LLM pristupnik, potpuno višejezičan, s kontrolom troškova nad
> *kupljenim* i *samostalno hostovanim* tokenima, te modernim interfejsom.

Die Eine Kette objedinjuje proizvoljan broj LLM provajdera (OpenAI, Anthropic, Gemini,
Azure, AWS Bedrock, Ollama, DeepSeek, Mistral, lokalni modeli …) iza **jednog**
OpenAI-kompatibilnog `/v1` interfejsa, uz B2B sloj za upravljanje iznad (preduzeća,
odjeli, korisnici, budžeti).

> 🚧 Javna alfa — vidi [`docs/`](./docs). Pokretanje: `docker compose up`.
