# ⛓️ Die Eine Kette (DieEineKette)

> ⚠️ **Javna alfa.** To je **samo brezplačni, javni del** projekta Die Eine Kette in je **v
> aktivnem razvoju** — pričakujte napake, nedokončane funkcije in spremembe. **Še ni primerno
> za produkcijo.** Povratne informacije in prijave napak so dobrodošle.

> **Ena vrata, da jih vse poveže.**
> Večnajemniški (B2B) LLM prehod, popolnoma večjezičen, z nadzorom stroškov nad
> *kupljenimi* in *samostojno gostovanimi* žetoni ter sodobnim vmesnikom.

Die Eine Kette združuje poljubno število ponudnikov LLM (OpenAI, Anthropic, Gemini,
Azure, AWS Bedrock, Ollama, DeepSeek, Mistral, lokalni modeli …) za **enim**
OpenAI-združljivim vmesnikom `/v1`, nad njim pa je B2B upravljalni sloj (podjetja,
oddelki, uporabniki, proračuni).

> 🚧 Javna alfa — glej [`docs/`](./docs). Zagon: `docker compose up`.
